Flexbox 속성 데모

1. order 속성

order 속성은 플렉스 아이템의 배치 순서를 결정합니다. 기본값은 0이며, 값이 작을수록 앞에 배치됩니다.

기본 순서 (order: 0)

A (order: 0)
B (order: 0)
C (order: 0)

order 속성 적용

A (order: 3)
B (order: 1)
C (order: 2)

음수 order 값 적용

A (order: 0)
B (order: -1)
C (order: 1)

2. flex-basis 속성

flex-basis 는 flex 아이템의 초기 크기를 설정합니다. 아이템이 늘어나거나 줄어들기 전의 기본 크기를 지정합니다.

4. flex-shrink 속성

flex-shrink 는 컨테이너가 작아질 때 아이템이 얼마나 줄어들지 결정합니다. 값이 클수록 더 많이 줄어듭니다.

5. flex 축약 속성 (flex-grow, flex-shrink, flex-basis)

flex 는 flex-grow , flex-shrink , flex-basis 의 축약형입니다.

형식: flex: grow shrink basis;

flex: 1 (= flex: 1 1 0)

flex: 1
flex: 1

서로 다른 flex 값

flex: 1

flex: 1

다양한 flex 축약 속성

flex: 1 1 100px
flex: 2 0 100px
flex: 0 1 auto

요약

  • order : 플렉스 아이템의 배치 순서를 설정합니다.
  • flex-basis : 아이템의 초기 크기를 결정합니다.
  • flex-grow : 남은 공간을 얼마나 차지할지 결정합니다.
  • flex-shrink : 공간이 부족할 때 얼마나 줄어들지 결정합니다.
  • flex : 위 세 속성을 한 번에 설정합니다 (flex: grow shrink basis).

자주 사용되는 패턴:

flex: 1 - 모든 아이템이 같은 크기로 공간을 채웁니다.
flex: 0 0 auto - 아이템이 자신의 원래 크기를 유지합니다.
flex: 0 0 200px - 아이템이 고정 크기를 유지합니다.
