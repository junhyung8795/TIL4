Emmet — 기본

에디터에서 Emmet 축약을 입력하고 Tab을 누르면 아래와 같은 HTML이 생성됩니다.

0) 문서 기본 구조

Emmet: !  (또는 html:5)
결과 (요약)
<!DOCTYPE html>
<html lang="en">
  <head>...</head>
  <body>...</body>
</html>

1) id / class

Emmet: div#my-id   /   span.my-class
div#my-id → id="my-id"
span.my-class → class="my-class"

2) 자식( > ) / 형제( + )

Emmet: main>section    /    main>section+aside
자식 section
형제 section
형제 aside

3) 반복( * ) / 번호( $ ) / 텍스트( { } )

Emmet: ul>li.item$*3{아이템 $}
  • 아이템 1
  • 아이템 2
  • 아이템 3

4) 리스트+링크 빠르게

Emmet: ul>li*3>a[href="#"]{Menu $}
  • Menu 1
  • Menu 2
  • Menu 3
