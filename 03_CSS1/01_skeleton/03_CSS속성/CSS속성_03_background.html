CSS 배경 속성 가이드

웹 디자인에서 요소의 배경을 스타일링하기 위한 다양한 CSS 배경 속성들을 알아봅니다.

background-color

요소의 배경색을 지정합니다. 색상 이름, HEX, RGB, RGBA, HSL, HSLA 등의 다양한 색상 값을 사용할 수 있습니다. 그라데이션은 background-image 속성을 사용하는 것이 더 일반적입니다.

1) 빨간색 (#FF6B6B)

2) 녹색 (#4ECDC4)

녹색 배경

3) 파란색 (#6C63FF)

파란색 배경

4) 그라데이션

그라데이션 배경

background-image

요소의 배경 이미지를 지정합니다. URL을 통해 외부 이미지를 사용하거나 그라데이션 함수를 통해 그라데이션 배경을 생성할 수 있습니다. 여러 이미지를 쉼표로 구분하여 다중 배경도 가능합니다.

1) URL 이미지

URL 이미지 배경

3) 방사형 그라데이션

background-repeat

배경 이미지의 반복 방식을 지정합니다. 가로, 세로로 반복하거나, 둘 중 하나의 방향으로만 반복하거나, 반복하지 않을 수 있습니다.

1) repeat (기본값)

repeat

2) repeat-x

repeat-x

3) repeat-y

repeat-y

4) no-repeat

no-repeat

background-position

배경 이미지의 초기 위치를 지정합니다. 키워드(top, right, bottom, left, center), 길이 값(px, em), 또는 백분율(%)을 사용하여 위치를 지정할 수 있습니다.

1) center

2) top right

3) bottom left

4) 75% 25%

background-size

배경 이미지의 크기를 지정합니다. auto(원본 크기), cover(요소를 완전히 덮음), contain(요소 내에 이미지가 모두 표시되도록), 또는 직접 크기 값을 지정할 수 있습니다.

1) auto (기본값)

auto

2) cover

cover

3) contain

contain

4) 50% 75%

50% 75%

background-clip

배경이 요소의 어느 영역까지 확장될지 지정합니다. border-box(테두리까지), padding-box(패딩까지), content-box(내용 영역만), text(텍스트만)의 옵션이 있습니다.
