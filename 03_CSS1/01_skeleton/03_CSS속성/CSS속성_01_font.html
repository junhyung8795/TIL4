font-style

텍스트의 스타일을 지정합니다. normal은 기본 스타일, italic은 이탤릭체, oblique는 기울임체를 나타냅니다. italic과 oblique는 비슷해 보이지만 구현 방식이 다릅니다.

font-variant

텍스트의 변형을 지정합니다. small-caps는 소문자를 작은 대문자로 표시하는 기능입니다. 주로 제목이나 헤더에 사용되어 텍스트에 독특한 스타일을 부여합니다.

1) normal

Hello! Text with normal letters.

2) small-caps

Hello! Text with small caps.

font-weight

텍스트의 두께(굵기)를 지정합니다. normal(400)과 bold(700) 같은 키워드나 100부터 900까지의 100단위 숫자 값을 사용할 수 있습니다. 숫자가 클수록 더 두꺼워지며, 각 값은 Thin(100), Extra Light(200), Light(300), Regular(400), Medium(500), Semi Bold(600), Bold(700), Extra Bold(800), Black(900)과 같은 두께를 나타냅니다. 단, 모든 폰트가 9가지 두께를 모두 지원하는 것은 아닙니다.
